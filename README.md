🤖 LoyaSense Engine: Agentic Financial Retention Intelligence
LoyaSense is an enterprise-grade AI engine built for Financial Institutions (Banks, SACCOs, and MFIs). It identifies high-risk customers through behavioral velocity analysis and automates the creation of personalized retention interventions using generative AI.

🚀 Key Features
Multi-Institution Predictive Modeling: Analyzes deposit-to-withdrawal ratios and engagement patterns applicable to any retail banking or credit union dataset.

Agentic Intervention Generation: Powered by Mistral AI to generate 1-on-1 personalized retention offers (Special Interest Rates, Loyalty Loans, or Fee Waivers).

Self-Service Analytics Portal: Simple CSV upload interface for branch managers and marketing teams to get instant, actionable insights.

Risk-Ranked Visual Hierarchy: Automated priority sorting with "Critical," "High Risk," and "Monitor" status indicators.

🛠️ Tech Stack
Language: Python 3.x

AI/LLM: Mistral AI API (Agentic Reasoning)

ML Stack: Scikit-Learn (Random Forest), Pandas (Data Engineering), Joblib

Web Framework: Flask

Environment: Render (Cloud Infrastructure)

📊 Business Logic & Value
The engine solves the "Silent Churn" problem—where customers slowly drain accounts before closing them.

Retention Yield: High-probability churners are identified 30-60 days before account closure.

Operational Efficiency: Eliminates the need for manual data sorting by marketing teams.

Hyper-Personalization: Each SMS/Email offer is context-aware based on the customer's specific engagement score.
